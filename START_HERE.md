# VideoGenAI - Local AI Video Generation

## Quick Start

### Option 1: Double-click to start
- Double-click `start.bat`

### Option 2: PowerShell (Recommended)
- Right-click `start.ps1` -> "Run with PowerShell"

### Option 3: Manual
```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

## Requirements
- Windows 10/11
- Python 3.10+
- NVIDIA GPU with 8GB+ VRAM

## Models
- Wan2.1-T2V-1.3B (8GB VRAM, recommended)
- Wan2.1-T2V-14B (24GB VRAM, best quality)
